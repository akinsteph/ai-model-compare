API Documentation

Endpoints

1. Get Current Pricing

GET /api/pricing

Returns the current pricing information for all AI models.

Example Response:
{
    "last_updated": "2024-11-10T14:00:00.000Z",
    "providers": {
        "openai": {
            "gpt4": {
                "input": "$0.03/1K tokens",
                "output": "$0.06/1K tokens"
            },
            "gpt35": {
                "input": "$0.0015/1K tokens",
                "output": "$0.002/1K tokens"
            }
        }
        // ... other providers
    }
}

2. Refresh Pricing

GET /api/refresh-pricing

Triggers an immediate refresh of the pricing data.

Example Response:
{
    "status": "success",
    "data": {
        // Updated pricing data
    }
}

Rate Limits

  • Pricing endpoint: 60 requests per minute
  • Refresh endpoint: 1 request per minute

Example Usage

curl https://your-domain.com/api/pricing
